E-commerce Price Prediction (Machine Learning Project)
Project Overview

This project focuses on analyzing an e-commerce dataset and building a machine learning model to predict the total order value (Total Price).
The project demonstrates a complete end-to-end workflow including data analysis, feature engineering, preprocessing, and model building.

Objectives
Perform Exploratory Data Analysis (EDA)
Handle missing values and data types
Detect and reduce skewness in data
Apply feature engineering techniques
Build and evaluate a Linear Regression model

Steps Performed
1. Data Cleaning
Handled missing values
Corrected data types
2. Exploratory Data Analysis (EDA)
Visualized distributions using histograms and boxplots
Identified skewness and outliers
Analyzed feature relationships using correlation heatmap
3. Feature Engineering
Created new meaningful features
Applied square root transformation to reduce skewness in target variable
4. Data Preprocessing
Train-test split
Scaling using StandardScaler
Handling missing values using SimpleImputer
5. Model Building
Implemented Linear Regression model
Built preprocessing pipeline
6. Model Evaluation
Evaluated using:
R² Score
Mean Absolute Error (MAE)
Root Mean Squared Error (RMSE)

Key Insights
Skewed data can negatively impact model performance
Feature engineering plays a crucial role in improving accuracy
Proper preprocessing helps avoid data leakage

Technologies Used
Python
Pandas
NumPy
Matplotlib & Seaborn
Scikit-learn
